# Code of Conduct

This project adopts the **Contributor Covenant v2.1**.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

Applies to issues, pull requests, discussions, commit messages, and any other interaction in spaces representing this project.

## Reporting

Report concerns privately via this repo's [security advisory flow](https://github.com/brko7/reservoarr/security/advisories/new) — the same channel as security issues, both go directly to the maintainer (`@brko7`). Do not file public issues for conduct reports.

## Enforcement

The maintainer reviews every report and decides on the response (warning, temporary or permanent ban from project spaces), referencing the Contributor Covenant's enforcement ladder. Decisions are final.
